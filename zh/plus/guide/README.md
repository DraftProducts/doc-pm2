# 指南

本指南将在几分钟内使您的node.js应用通过PM2 Plus进行监控.

### 术语

我们来解释一下在本指南中使用的一些术语:

**bucket** 是我们用来收集多个服务器的实体。 一个bucket通常用于从一个项目中收集多个服务器.

**server** 是PM2 Plus从其接收来自一个或多个进程信息的实体.

**process** 是进程列表 (`pm2 ls`)的实体。
